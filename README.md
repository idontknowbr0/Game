## UMD Game Dev Club Game Jam Project – Wizard Survivor
Overview:
UMD Game Development Club에서 진행된 Game Jam에 팀으로 참가하여 개발한 2D 액션 서바이벌 스타일 게임입니다.
Vampire Survivors에서 영감을 받아, 끊임없이 등장하는 적을 피하고 처치하며 생존하는 핵심 재미 요소를 구현하는 데 집중했습니다.

Features
- 자동 공격 기반 서바이벌 전투 시스템
- 점진적으로 증가하는 적 스폰 시스템
- 간단한 플레이어 이동 및 회피 메커니즘
- 짧은 시간 내 완성 가능한 Game Jam 스타일 구조

Team Project
- UMD Game Dev Club Game Jam 참가 프로젝트
- 팀 기반 개발 및 역할 분담 경험
- 제한된 시간 내 기능 중심 개발 및 협업 진행
